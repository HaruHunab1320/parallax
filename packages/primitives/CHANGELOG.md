# Changelog

## [0.1.0] - 2026-03-11

### Added
- Initial release of the Parallax primitives library.
- 32 primitives across 12 categories: execution, aggregation, confidence, control, coordination, transformation, workflow, temporal, resource, transaction, event, and goal.
- `PatternComposer` class for analyzing requirements and composing primitives into patterns.
- `PatternAssembler` class for converting composed patterns into executable Prism code.
- `PatternValidator` class for validating pattern structure, connections, and confidence propagation.
- Core types: `OrchestrationRequirements`, `Primitive`, `ComposedPattern`, `ExecutablePattern`, `PatternValidation`.
